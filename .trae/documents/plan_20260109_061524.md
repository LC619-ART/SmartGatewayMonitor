# 实现基于 Flask 的前端网页应用

## 步骤概述
1. **创建前端应用文件**：在 `d:\Project\kehu` 目录中创建 `frontend.py` 文件，包含完整的 Flask 应用代码
2. **安装依赖**：使用 pip 安装 Flask 包
3. **运行前端服务**：启动 Flask 开发服务器
4. **验证访问**：通过浏览器访问前端页面

## 详细实现
1. **创建 frontend.py 文件**：
   - 实现 TCP 客户端连接云服务器
   - 接收并解析服务器转发的数据
   - 提供 HTTP 接口展示数据和接收控制指令
   - 自动生成 HTML 模板页面

2. **安装依赖**：
   - 执行 `pip install flask` 安装 Flask 框架

3. **运行前端服务**：
   - 执行 `python frontend.py` 启动服务
   - 服务将在 `http://localhost:5000` 运行

4. **前端页面功能**：
   - 显示本地温湿度数据（来自网关）
   - 显示实时天气数据（来自 API）
   - 显示高温报警提示
   - 提供开窗、关窗、自动模式控制按钮
   - 每 3 秒自动刷新数据

## 技术特点
- 使用 Flask 框架构建轻量级 web 应用
- 采用 TCP 协议与云服务器通信
- 多线程处理数据接收，确保实时性
- 响应式设计，页面自动刷新
- 错误处理和重连机制，提高稳定性
